Project #4 Experimental Clock

Design Process

My design process was to first create sketches in Illustrator and then trying to use a random generator for circles and creating variables that would allow it to stay for 50 seconds and have another circle appear that says "Time to Cook" once the 50 seconds are over. I used framerate to determine how long it would take for 200 circles to appear.



Reflections

I wanted to show the way chickpeas is cooked in my household. We soak chickpeas a whole day before cooking with it. I wanted to show this using a fairly short timer. I think this project helped me understand the if/else function and I also played around with the map function so that the text moves with the big circle.
